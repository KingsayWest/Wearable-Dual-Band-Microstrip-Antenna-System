# Wearable Dual-Band Microstrip Patch Antenna System

## Overview

This repository presents the complete design and simulation of a wearable dual-band microstrip patch antenna integrated with an RF transceiver and ambient RF energy harvesting system for sports tracking, biomedical monitoring, and Wireless Body Area Network (WBAN) applications.

The antenna simultaneously operates at:

• GPS L1 Band (1.575 GHz)

• ISM Band (2.45 GHz)

The project was designed and analyzed using ANSYS HFSS 2024 R2.

# Simulation Workflow

1. Design the antenna geometry

2. Define Rogers RO3003 substrate

3. Add U-slot

4. Configure microstrip feed

5. Set radiation boundary

6. Define wave port

7. Configure solution setup

8. Run adaptive mesh

9. Perform frequency sweep

10. Optimize antenna dimensions

11. Simulate SAR

12. Generate radiation patterns

13. Export results

# Simulation Results

The antenna successfully achieved

✔ Resonance at 1.575 GHz

✔ Resonance at 2.45 GHz

✔ Return Loss below −20 dB

✔ Peak Gain of 4.37 dBi

# Wearable-Dual-Band-Microstrip-Antenna-System
Design and simulation of a wearable dual-band microstrip patch antenna operating at GPS L1 (1.575 GHz) and ISM (2.45 GHz) with integrated RF transceiver and ambient RF energy harvesting for sports, biomedical, and Wireless Body Area Network (WBAN) applications using ANSYS HFSS 2024 R2.
